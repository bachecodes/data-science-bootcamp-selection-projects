# Superstore Data Analysis  Pandas

## Project Overview
This project analyzes the Superstore dataset using Python Pandas. The goal is to explore sales, profit, and customer behavior.

## Key Tasks
- Data cleaning and preprocessing
- Feature engineering (date-based features)
- Aggregation and grouping
- Pivot table analysis
- Data visualization

## Key Insights
- Identified most profitable categories and regions
- Observed trends in sales over time
- Found relationships between sales and profit

## Tools Used
- Python
- Pandas
- Matplotlib
